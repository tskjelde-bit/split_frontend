# Prosjekt: Innsikt — Boligmarkedet i Oslo

## Typografi
Følg reglene i `docs/TYPOGRAPHY.md` for all typografi på hele nettsiden.
Alle font-sizes skal bruke rem, ikke px. Ingen unntak.
Kun 3 font-weights: 400, 600, 700.
Uppercase kun på korte labels og knapper — aldri på overskrifter eller setninger.

## Farger
Følg reglene i `docs/COLORS.md` for alle farger på hele nettsiden.
Alle farger via CSS custom properties. Ingen hardkodede hex/rgb-verdier i komponenter.
Light mode er default (:root), dark mode via .dark-klasse på <html>.
4 bakgrunnsnivåer, 3 tekstnivåer, 2 border-nivåer. Ingen andre.
